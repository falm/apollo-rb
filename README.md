# Apollo-rb

携程Apollo配置中心 Ruby客户端

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'apollo-rb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install apollo-rb
